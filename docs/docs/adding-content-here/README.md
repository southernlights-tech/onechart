# Southern Lights Docs

Welcome to the documentation repository for Southern Lights. This repository is organized to help both internal teams and external collaborators quickly find the information they need regarding our company procedures, technical infrastructure, and overall business strategy.

## Repository Structure

```
.
├── README.md                # Overview of the repository, purpose, and navigation guide
├── docs/                    # General documentation
│   ├── company-overview.md  # Business overview, mission, values, etc.
│   ├── team.md              # Team and organizational structure
│   └── roadmap.md           # Product and business roadmap
├── procedures/              # Operational procedures and workflows
│   ├── onboarding.md        # New employee onboarding process
│   ├── customer-support.md  # Customer support procedures
│   └── escalation.md        # Escalation processes
└─── tech-docs/               # Technical documentation
    └── guides/              # Developer guides, installation, and troubleshooting
        ├── setup.md         # Environment setup instructions
        └── troubleshooting.md # FAQ and common issue resolutions
```

## Getting Started

1. **Clone the Repository**  
   To get started, clone the repository to your local machine:
   ```bash
   git clone git@github.com:southernlights-tech/docs.git 
   ```

2. **Explore the Documentation**  
   Navigate through the folders to find the relevant documents:
   - **docs/**: High-level business and strategic documents.
   - **procedures/**: Operational procedures that guide day-to-day processes.
   - **tech-docs/**: All technical documentation including architecture, API details, and developer guides.

3. **Contributing**  
   We welcome contributions! If you want to help improve our documentation:
   - Create a new branch with your updates.
   - Submit a pull request with a clear description of your changes.

4. **Feedback & Support**  
   For any questions or feedback regarding the docs, please open an issue or contact the documentation team at [info@southernlights.tech](mailto:info@southernlights.tech).
